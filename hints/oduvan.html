Sure. Here is what you need to know.

To initialize an empty list do this:

new_list = []

To add an item into the list do this:

new_list.append(item)

To return the list use this:

return new_list

So the correct answer might look like this:

def checkio(data):
    result = []
    for item in data:
        if data.count(item) > 1:
            result.append(item)
    return result

I would also recommend you go through all available list methods, so you know what other methods you can use in future.

You can simply replace code 

def checkio(data):
    result = []
    for item in data:
        if data.count(item) > 1:
            result.append(item)
    return result

To return the list use this:

def checkio(data):
    return filter(lambda a: data.count(a) > 1, data)

It is shorter and even easier to read.
